Projeto ADS

Sistema de Compartilhamento de Materiais Acadêmicos

```

Lucas Matias

Curso de Análise e Desenvolvimento de Sistemas

```

Problema

```

Muitos estudantes têm dificuldade para encontrar materiais de estudo organizados e confiáveis.

```

Objetivos

```
  • Criar uma plataforma colaborativa
  • Compartilhar PDFs e vídeos
  • Facilitar o acesso ao conhecimento
  • Melhorar a experiência dos estudantes
```

Tecnologias Utilizadas

```
  • HTML5
  • CSS3
  • JavaScript
  • MySQL
  • GitHub
```

Protótipo

``` ```

Benefícios

```
  • Acesso rápido aos materiais
  • Comunidade colaborativa
  • Organização por disciplinas
  • Maior produtividade acadêmica
```

Conclusão

```

A plataforma pode contribuir para melhorar o aprendizado e a troca de conhecimento entre estudantes.
```
